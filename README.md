# Xadrez
Xadrez em C com OOP (Structs/Ponteiros) e Persistência de Dados.  Trabalho de Estrutura de Dados (UNICESUMAR). Inclui: Sistema de Login com senha obscurecida; salvamento de estatísticas de partidas (V/D/E) em TXT; modo convidado; resgate de jogos não finalizados; e ranking de estatísticas. Implementação totalmente baseada em structs e ponteiros
